# Shared Media Drop Folder

Drop images, videos, and other media files here for Antigravity to use.

## How to use
1. Copy/paste or drag files into this folder via RDP
2. Tell Antigravity what to do with them (e.g. "Use the logo I dropped in shared/ for the hero section")
3. Antigravity will move/optimize/embed them as needed

## Naming tips
- Use descriptive names: `team-photo-adam.jpg` not `IMG_4392.jpg`
- Lowercase with dashes: `hero-banner.png`
- Include dimensions if relevant: `logo-400x200.svg`

## Supported formats
Images: `.jpg`, `.png`, `.svg`, `.webp`, `.avif`
Videos: `.mp4`, `.webm`
